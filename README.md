# Company Financial Trend Analysis

## 1. Problem & User
**Target user**: Small business owners or finance beginners  
**Problem**: Quickly view company revenue and profit trends without complex Excel operations

## 2. Data Source
- **Source**: Company financial records (sample data)
- **Fields**: Year, Revenue (10K RMB), Profit (10K RMB)
- **Time range**: 2020-2024

## 3. Methods
- **pandas**: Data loading, cleaning, and calculation (growth rate, profit margin, cumulative profit)
- **matplotlib**: Data visualization (trend charts, bar charts, growth rate comparison)

## 4. Key Findings
- Revenue grew from 1,000,000 to 1,850,000 RMB (+85%)
- Profit grew from 150,000 to 350,000 RMB (+133%)
- Profit margin increased from 15% to 18.9%

## 5. How to Run
1. Open `financial_analysis.ipynb` in Jupyter Notebook / ANANCODA
2. Run all cells from top to bottom
